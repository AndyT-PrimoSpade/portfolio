IBM NLU Analyzer

Project Description

Using ReactJS with IBM NLU to build a simple Sentiment Analyzer to detecte positive or negative sentiment in text. It can be used by businesses to detect sentiment in social data, gauge brand reputation, and understand customers.

Project Date Dec, 2021
Project Type Self Learning
How It Works Click on the link below, you will reach a simple page with a textbox. You can type in anything you want, when click on analyze sentiment it will return 3 result - "positive", "neutral", "negative". To get a more detail on the analyze you can click on analyze emotion to get the break down for the analyze. You might also input URL and it will read the page to analyze the text.
